# Smart Fridge System - Умный Холодильник

Система управления холодильником с удалённой настройкой температурных режимов, мониторингом и уведомлениями.

## Архитектура

```
┌─────────────────┐     MQTT      ┌─────────────────┐
│   Мобильное     │◄─────────────►│   Бэкенд        │
│   приложение    │   WebSocket   │   (Node.js)     │
└─────────────────┘               └────────┬────────┘
                                           │
                                    ┌──────▼──────┐
                                    │   MQTT      │
                                    │   Broker    │
                                    │  (Mosquitto)│
                                    └──────┬──────┘
                                           │
                                    ┌──────▼──────┐
                                    │   ESP32     │
                                    │  (Холодиль- │
                                    │   ник)      │
                                    └─────────────┘
```

## Компоненты

### 1. Железо (Hardware)
- **Микроконтроллер**: ESP32 DevKit v1
- **Датчики температуры**: DS18B20 (2 шт. - холодильник/морозилка)
- **Реле**: 2-канальное для управления компрессором и светом
- **Дисплей**: OLED 0.96" I2C (локальное отображение)
- **Кнопки**: 3 шт. для локального управления

### 2. Прошивка (Firmware)
- Платформа: PlatformIO / Arduino
- Протокол связи: MQTT
- Функции: чтение датчиков, управление реле, отправка телеметрии

### 3. Бэкенд (Backend)
- Node.js + Express
- MQTT клиент для связи с устройствами
- WebSocket для реального времени
- База данных: SQLite/PostgreSQL
- REST API для мобильного приложения

### 4. Фронтенд (Frontend)
- React Native / Flutter (мобильное приложение)
- Веб-интерфейс на React

## Возможности

✅ Удалённая настройка температурных режимов  
✅ Мониторинг температуры в реальном времени  
✅ Уведомления о критических изменениях  
✅ История температурных показателей  
✅ Локальное управление через кнопки  
✅ Энергосберегающий режим  

## Быстрый старт

1. Настройте оборудование согласно схеме в `hardware/`
2. Загрузите прошивку из `firmware/`
3. Запустите брокер сообщений и бэкенд
4. Подключите мобильное приложение

Подробная документация в папке `docs/`.
