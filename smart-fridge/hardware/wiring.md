## Схема подключения

### Датчики температуры DS18B20

```
Датчик 1 (Холодильник)          Датчик 2 (Морозилка)
    VCC ──────────┐                  VCC ──────────┐
    GND ──────────┼── GND ESP32    GND ──────────┼── GND ESP32
    DATA ───┬─────┘                DATA ───┬─────┘
            │                              │
         4.7кОм                         4.7кОм
            │                              │
         3.3V                           3.3V
            │                              │
         GPIO 4                         GPIO 16
```

### Реле модуль

```
Реле 1 (Компрессор)     Реле 2 (Свет)
    VCC ───────────────── 5V ESP32
    GND ───────────────── GND ESP32
    IN1 ───────────────── GPIO 17
    IN2 ───────────────── GPIO 18
    
Нагрузка:
    COM ───────────────── Фаза 220В
    NO  ───────────────── Компрессор/Свет
```

### OLED дисплей SSD1306

```
OLED Display
    VCC ───────────────── 3.3V ESP32
    GND ───────────────── GND ESP32
    SCL ───────────────── GPIO 22 (I2C SCL)
    SDA ───────────────── GPIO 21 (I2C SDA)
```

### Кнопки управления

```
Кнопка 1 (Mode)        Кнопка 2 (+)         Кнопка 3 (-)
    ┌───┐                  ┌───┐                  ┌───┐
    │   │                  │   │                  │   │
    └─┬─┘                  └─┬─┘                  └─┬─┘
      │                      │                      │
   10кОм                   10кОм                   10кОм
      │                      │                      │
   3.3V                   3.3V                   3.3V
      │                      │                      │
   GPIO 5                 GPIO 19                GPIO 20
```

### Индикатор статуса

```
LED (+) ─── 220Ω ─── GPIO 23
LED (-) ─── GND ESP32
```

## Питание

```
Блок питания 5В 2А
    ├── 5V ─── VIN ESP32
    ├── 5V ─── VCC реле
    └── 5V ─── Преобразователь 3.3V для датчиков
```

## Меры безопасности

⚠️ **ВАЖНО**: При работе с напряжением 220В соблюдайте технику безопасности!

1. Используйте качественную изоляцию для высоковольтных соединений
2. Размещайте реле в закрытом корпусе
3. Не допускайте попадания влаги на электронику
4. Используйте предохранитель на входе 220В
5. Заземлите корпус холодильника

## Корпус

Рекомендуется разместить электронику в герметичном корпусе IP65 вне охлаждаемой зоны, с выводом датчиков внутрь камер.
